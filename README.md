# Исследование стартапов

## Описание проекта

Подготовить и проанализировать объединённый набор исторических данных о стартапах для выявления закономерностей, влияющих на успешность сделок по их покупке, 
а также для ответа на ключевые вопросы заказчика о достоверности и значении показателей.

## Этапы проекта

1. Знакомство с данными: загрузка и первичная предобработка.
2. Предобработка данных, предварительное исследование.
3. Исследовательский анализ объединённых таблиц.
4. Итоговые выводы.

## Навыки и инструменты

**Навыки**
1. Подготовка данных.
2. Аналитическая проверка.
3. Выявление бизнес-инсайтов.
4. Визуализация и отчётность.

**Инструменты**  
`Jupyter Notebook`, `Python`.  
**Библиотеки**  
`pandas`, `numpy`, `matplotlib.pyplot`, `seaborn`, `scipy`, `matplotlib.ticker`, `matplotlib-venn -q`.

## Вывод
Финансирование стартапов распределено крайне неравномерно: большинство привлекают скромные суммы, тогда как единицы компаний обеспечивают сверхприбыль.  
Для заказчика это означает необходимость сочетания **стратегии надёжных вложений** с **готовностью к рисковым сделкам**, которые могут принести кратный рост.
